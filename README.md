# Currency_Prediction_Project
This repository showcases the process of coding a Streamlit app to predict future exchange rates for 22 different currencies using a range of different programs:

(Emoji here)Disclaimer: Google Collab was used to code, analyse and save the models for each currency, however some contents have been rewritten into Jupyter Notebook for Github compatibility.


1. The (post as link)Analysis folder contains the separate Jupyter Notebook files showcasing the code used to visually explore the dataset for three modelling techniques:
- LSTM Modelling
- ARIMA/SARIMA
- Prophet Models
Evaluation of these models are also calculated using MAE, MAPE and RMSE metrics.

2. The (link to excel file) file records the evaluation metrics and highlights which model was chosen to best represent each currency

3. The (link) Models folder contains the saved models for each currency. 

4. The (link to VSCode file) file showcases the development of the Streamlit application including a virtual environment created using Python version 3.10 to install and import packages such as Tensorflow.
